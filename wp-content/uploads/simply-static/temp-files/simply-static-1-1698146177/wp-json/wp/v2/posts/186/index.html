CNM is a powerful Python package tailored for computational neuroscience modeling. With its user-friendly design and flexibility, it serves as an invaluable tool for constructing and running simulations of neural networks and other computational models in the realm of neuroscience. It is designed to be a GUI simulator later, letting researchers use and infer various models available.
There are a plethora of models available. We will be using some of the most common models out there.
<\/p>\n\n\n\n

How to use it?<\/h2>\n\n\n\n

The project will soon be available in Pypi<\/a><\/mark>. Since it’s in the early progress, few models will be available and constantly get updated\/added. You can install it using git<\/mark> for now.
git clone https:\/\/github.com\/Da4kek\/CNM.git<\/mark><\/code><\/p>\n\n\n\n

Usage:<\/h3>\n\n\n\n

cd CNM<\/mark><\/code>
The repository’s README.md<\/mark> will give the theoretical aspects of the project, including the classification of models and hierarchy of the project.
Check out the Example_usage.ipynb<\/a><\/mark> for reference.<\/p>\n\n\n\n

src.algorithms:<\/h3>\n\n\n\n
Contains algorithms necessary to preprocess the data\/input (optional).
usage: from src.algorithms import *<\/code><\/p>\n\n\n\n
src.models:<\/h3>\n\n\n\n
Contains model classified into 3 categories according to D.Marr<\/a><\/mark> (Descriptive, Interpretive, and Mechanistic).
usage: from src.models import *<\/code> 
Descriptive.NRM, Descriptive.PRM and Descriptive.NFM<\/code> (Stands for Neural firing rate models, Population rate models, and Neural field models).
Interpretive.IFM and Interpretive.GLM<\/code> (Stands for Integrate-and-Fire models, and Generalized linear models).
Mechanistic.HH, Mechanistic.SYN, Mechanistic.BIO<\/code> (Stands for Hodgkin-Huxley model, Synaptic models, and Biophysical models).
Each model is different in their mathematical and theoretical ways.<\/p>\n\n\n\n
src.utils:<\/h3>\n\n\n\n
Contains utility functions such as plotting and other constant functions (optional). 
usage: from src.utils import *<\/code><\/p>\n\n\n\n
